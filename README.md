# Qui sommes-nous?

SoftFluent est une ESN experte en développement sur les technologies Microsoft élue au Palmarès Great Place To Work en 2016 et dans le Top des ESN FrenchWeb qui recrutent. 

L'entreprise est à la pointe de l’innovation sur les technologies Microsoft : C#, Azure, ASP.NET, MVC, WPF, SQL Server, TypeScript, Xamarin et des technologies comme Unity, Angular et React.

Les développeurs de SoftFluent interviennent pour tout type de clients sur les sujets suivants : 
- Conseil et Expertise
- Développement d'applications
- Innovation : Chatbots, Réalité Virtuelle (VR), Cognitive Services 

Devenez-vous aussi un(e) SoftFluentee et venez faire grandir la boîte grâce à vos initiatives !
